# 🚴 Ciclismo App

Una aplicación web para la gestión de eventos de ciclismo.

## 🛠️ Stack Tecnológico

- **Frontend**: React + Vite + Bootstrap
- **Backend**: Node.js + Express + Sequelize  
- **Base de datos**: PostgreSQL
- **Autenticación**: JWT

## 🚀 Instalación

### Prerrequisitos
- Node.js 18+
- PostgreSQL 16+
